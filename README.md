# Cyber Security Internship Tasks (CodeAlpha)

This repository contains my completed tasks for the CodeAlpha Cyber Security Internship. These projects are developed using Python and demonstrate basic cybersecurity concepts such as network traffic monitoring, password strength checking, and port scanning.

## Tasks Included

### Task 1: Network Packet Sniffer

This program captures live network packets and displays important information such as source IP address, destination IP address, and protocol type (TCP, UDP, ICMP) using the Scapy library.

### Task 2: Password Strength Checker

This program checks whether a password is strong, medium, or weak based on security rules like password length, uppercase letters, lowercase letters, numbers, and special characters.

### Task 3: Port Scanner

This program scans common network ports (80, 443, 445, 3389) on a target IP address and identifies whether the ports are open or closed using Python’s socket module.

## Tools & Technologies Used

* Python
* Scapy Library
* Socket Module
* Regular Expressions
* VS Code
