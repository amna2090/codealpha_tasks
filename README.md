Cyber Security Python Projects

This repository contains basic cybersecurity projects developed using Python. These projects demonstrate concepts such as network packet sniffing, password strength checking, and port scanning.
Tasks Included
Task 1: Network Packet Sniffer

Captures live network packets and displays source IP, destination IP, and protocol type using Scapy.

Task 2: Password Strength Checker

Checks password strength based on length, uppercase letters, lowercase letters, numbers, and special characters.

Task 3: Port Scanner

Scans common ports (80, 443, 445, 3389) on a target IP address and shows whether they are open or closed.

Tools Used

Scapy
